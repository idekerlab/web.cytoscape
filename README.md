# _web.cytoscape_ Prototype

## Introduction
This is a new prototype of web.cytoscape.org web site.

## How to use

### Install Dependencies

```
$ npm install
```

### Run with local test server

```
$ npm start
```

### Build

```
$ npm run build
```

### Cleanup

```
$ npm run clean
```
